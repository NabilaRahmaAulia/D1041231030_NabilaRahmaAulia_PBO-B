1.	Saya memahami bahwa Naming Convention adalah aturan atau standar dalam penamaan variabel, class, method, dan elemen lain dalam pemrograman 
agar kode menjadi rapi, konsisten, dan mudah dipahami. Hal ini penting untuk meningkatkan keterbacaan kode (readability), 
memudahkan pemeliharaan kode (maintainability), memfasilitasi kolaborasi tim, mengurangi kesalahan interpretasi, mencerminkan profesionalisme developer. 
Variabel di Java menggunakan camelCase convention, artinya variabel dimulai dengan huruf kecil, kata pertama ditulis lowercase, dan 
kata berikutnya dimulai dengan huruf kapital, tidak boleh menggunakan underscore atau karakter khusus lainnya, 
nama harus deskriptif dan mencerminkan tujuan variabel.

3.	Dalam Java, terdapat operator aritmatika dasar (+,-,*,/,%), operator increment dan decrement (++x, x++, --x, x--), compound assignment operators (+=, -=, *=, /=, %=), operator precedence atau prioritas (++ -, * / %, + -, = += = *= /= %=, {}). Lalu, ada Type Casting dan Konversi, yaitu proses mengubah tipe data dari satu jenis ke jenis lainnya. Ini memiliki dua jenis, yaitu Widening (konversi otomatis dari tipe data yang lebih kecil ke tipe data yang lebih besar) dan Narrowing (konversi dari tipe data yang lebih besar ke tipe data yang lebih kecil, perlu explicit cast dan berisiko kehilangan data).
4.	Contoh penerapan:
int panjang = 10;
int lebar = 5;
int luas = panjang * lebar;
System.out.println("Luas = " + luas);
Kode tersebut merupakan program sederhana dalam Java untuk menghitung luas persegi panjang. 
Pada baris pertama, int panjang = 10; dibuat sebuah variabel bertipe data int (bilangan bulat) bernama panjang yang diberi nilai 10. 
Pada baris kedua, int lebar = 5; dibuat variabel bernama lebar dengan nilai 5. 
Selanjutnya, pada baris int luas = panjang * lebar;, program melakukan operasi perkalian antara nilai panjang dan lebar, yaitu 10 Ã— 5, 
sehingga menghasilkan nilai 50 yang kemudian disimpan ke dalam variabel luas. 
Terakhir, perintah System.out.println("Luas = " + luas); digunakan untuk menampilkan hasil perhitungan ke layar. 
Teks "Luas = " digabungkan dengan nilai variabel luas menggunakan tanda +, sehingga output yang ditampilkan adalah Luas = 50.
